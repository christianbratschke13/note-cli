# note-cli

A lightweight command-line note-taking tool built with Rust.

---

## Features

- Create notes directly from your terminal
- List all existing notes
- Read notes instantly
- Delete notes you no longer need
- Store notes as simple files

---

## Quick Start

### Clone the repository

```bash
git clone https://github.com/christianbratschke13/note-cli.git
cd note-cli
```

### Build the project

```bash
cargo build --release
```

### Run the application

```bash
cargo run -- add "My first note"
cargo run -- list
cargo run -- delete <hash>
```

---

## Why note-cli?

note-cli is a small learning project focused on:

- Rust fundamentals
- File system operations
- Command-line interfaces
- Error handling

It aims to provide a simple and fast note-taking experience directly from the terminal.
